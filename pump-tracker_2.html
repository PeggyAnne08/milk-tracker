Connect Google Sheets

To save your data directly to your spreadsheet, you'll need a Google Apps Script web app URL. Here's how to set it up:

  1. Open your Google Sheet → click Extensions → Apps Script
  2. Delete any existing code and paste in the script from the instructions below
  3. Click Deploy → New deployment → Web app
  4. Set "Who has access" to Anyone → click Deploy
  5. Copy the web app URL and paste it here 👇
